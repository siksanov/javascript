--------------------------------------------------------------------------------------------
1. Состав архива
--------------------------------------------------------------------------------------------

Архив включает в себя следующие папки и файлы (вложенность папок и файлов показана
отступами):

 * <номер урока> - папка с материалами урока с указанным номером. Состав папки:

    - !sources - папка с исходными материалами, необходимыми для выполнения упражнений в
      текущем уроке;

    - ex<номер раздела> - результаты выполнения упражнения, приведенного в разделе с
      указанным номером;

    - s<номер раздела> - результаты выполнения самостоятельных упражнений, приведенных
      в разделе с указанным номером;

 * readme.txt - этот текстовый файл.



--------------------------------------------------------------------------------------------
2. Подготовка, запуск и останов традиционного веб-приложения и бэкенда
--------------------------------------------------------------------------------------------

В папках ex29.1-ex29.5 и s29.6, находящихся в папке 29, хранится традиционное
веб-приложение, разработанное на уроке 29. Для его подготовки к запуску необходимо
выполнить действия, перечисленные далее.

 1. Скопировать папку photogallery, хранящую Node-проект бэкенда, из одной из упомянутых
    папок куда-либо на локальный диск.

 2. Запустить утилиту Terminal или командную строку.

 3. В этой утилите:

    3.1. Выполнить переход в копию папки photogallery.

    3.2. Установить необходимые для работы библиотеки, подав команду:

         npm install

Для запуска приложения достаточно запустить утилиту Terminal или командную строку,
в этой утилите перейти в копию папки photogallery и подать команду:

npm run serve

Чтобы остановить запущенное веб-приложение, достаточно переключиться в экземпляр утилиты
Terminal или командной строки, в котором оно было запущено, нажать комбинацию клавиш <Ctrl>+<C>
и согласиться с остановкой приложения, последовательно нажав клавиши <Y> и <Enter>.

В папках ex30.2 и ex30.4, вложенных в папку 30, хранится бэкенд, разработанный на уроке 30.
Его подготовка, запуск и останов выполняются так же.
